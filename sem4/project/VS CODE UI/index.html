About GradeAid

GradeAid is a software that takes in answer sheets and assignments as input (photos/text), extracts the text using OCR if it is a photo and then various ML techniques to extract to grade the answer, with the help of an answer key. This system can also provide feedback regarding which key points were & were not missing in the answer. This will help in grading and declaring results faster & unbiased manner.
